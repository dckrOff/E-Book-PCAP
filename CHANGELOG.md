# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

## [0.1.0] - 0026238

### Добавлено

- Инициализация проекта Android с использованием Kotlin
- Настройка базовой архитектуры MVVM
- Настройка Hilt для внедрения зависимостей
- Настройка Navigation Component для навигации между экранами
- Настройка Room для работы с базой данных
- Настройка Retrofit для работы с сетью
- Настройка Coroutines и Flow для асинхронных операций
- Настройка Material Design 3 для UI компонентов

### Изменено

- Обновлена структура проекта для соответствия Clean Architecture
- Оптимизирована структура пакетов для лучшей организации кода

### Исправлено

- Нет
